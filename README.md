# fasciste - personal site

My personal presentation website built with vanilla HTML, CSS and JavaScript.
Catppuccin Macchiato theme, terminal aesthetic, matrix rain background.

## Stack

- HTML / CSS / JS - no frameworks, no dependencies
- Fonts: [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) + [Syne](https://fonts.google.com/specimen/Syne) via Google Fonts

## Use it

This project is fully open source. Do whatever you want with it - fork it, modify it, use it as a base for your own site, strip it for parts. No credit required.
